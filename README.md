# gerador_de_primos
Algoritmo que, segundo o principio de que todo primeiro divisor de determinado inteiro é primo, encontra números divisiveis apenas por eles mesmos ou por 1. 
Isso é visto pelo fato de que, tida uma sequência de múltiplos, como: x, 2x, 3x, 4x [...], o menor valor sempre dividirá, com resto zero, todos os valores subsequentes.
